# Study TypeScript

Repositório dedicado ao estudo estruturado de TypeScript com foco em backend profissional.

## Objetivos

- Compreender tipagem estática
- Aprender interfaces e types
- Entender generics
- Trabalhar com decorators
- Aplicar TypeScript em Node.js

## Estrutura

- fundamentos → Tipos básicos e sintaxe
- tipos-avancados → Generics, Utility Types
- oop → Classes, interfaces, herança
- node-integration → Uso com Node.js
- arquitetura → Estruturação profissional com TS

## Meta

Utilizar TypeScript como padrão principal para projetos backend.
